------------------------------------מתורגם לאנגלית------------------------------------

Role: Senior Frontend Developer and Software Architect.

Objective: Develop a career recommendation application based on a 4-question questionnaire and scoring weights.

Stack: React, Vite, TypeScript, Tailwind CSS.

Data: Questions, professions, and weights will be managed in a separate JSON file and loaded at runtime.
Data Object:

JSON
{
  "questions": [
    {
      "id": 1,
      "text": "האם אתה אוהב לעבוד עם אנשים?",
      "weights": {
        "teacher": 5,
        "social_worker": 7,
        "graphic_designer": 0
      }
    }
  ],
  "professions": {
    "teacher": "מורה",
    "social_worker": "עובד סוציאלי",
    "graphic_designer": "גרפיקאי"
  }
}
Logic: A "Yes" answer adds points according to the weights; a "No" answer does not add points.

Result: Upon completion, the profession with the highest score will be displayed, along with an option to restart.
